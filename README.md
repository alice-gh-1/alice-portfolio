# AliceOS

Static GitHub Pages page for AliceOS.

Built as a tiny no-build HTML/CSS/JS site:
- fullscreen Unix-like boot animation
- interactive fake shell
- commands: `help`, `motd`, `ls`, `cat`, `reboot`, `status`, `clear`, `uptime`
- no UI chrome
- no secrets

Local files:
- `index.html`
- `style.css`
- `script.js`
