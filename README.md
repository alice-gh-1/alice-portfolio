# AliceOS

Static GitHub Pages page for AliceOS.

Built as a tiny no-build HTML/CSS/JS site:
- fullscreen Unix-like boot animation
- interactive terminal-style prompt
- commands: `help`, `motd`, `reboot`, `status`, `clear`
- no UI chrome
- no secrets

Local files:
- `index.html`
- `style.css`
- `script.js`
